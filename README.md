## Description:
This project is intended to create a social media API framework that allows the user to create, view, update and delete User and Thought objects. Additionally, the user can add friends and reactions to Users and Thoughts respectively. This project was made as part of the requirements for a web development bootcamp.

## Installation:
* Clone the git repository onto your system

## Usage:
* Open index.js in the integrated terminal
* Run "npm i" in the terminal
* Type "node index.js"
* Navigate to Insomnia to create queries
